TAFPLOTer

This is an application to generate TAFPLOTs in the style described by:

Saboisky JP, Butler JE, Walsh LD & Gandevia SC (2007). New display of the timing and firing frequency of single motor units. J Neurosci Methods 162, 287â€“292.

Currently it is very simple and poorly documented but it does work. Get in touch if you have problems, or fix them and submit a pull request.

The Igor Pro script used for the paper is in the source repository, so if that's all you want download the .zip or .tar.gz file.

Installation

Get the latest release binary from https://github.com/Tanglo/TAFPLOTer/releases and put the .app file where you want to run it from
