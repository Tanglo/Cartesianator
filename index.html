Cartesianator

An OS X that measures cartesian coordinates from images.

Download this project as a .zip file Download this project as a tar.gz file

TAFPLOTer

Cartesianator is used to measure calibrated cartesian coordinates of user selected points. Once calibration is complete Cartesianator will step though the specified images and extract the points in the images that use use clicks on. It is intended to be used to easily extract numerical data (i.e. coordinates in the image space) from still images.

Installation

There is no release binary available yet. This project is in very early stages.
